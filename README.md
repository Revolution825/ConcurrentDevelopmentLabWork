# ConcurrentDevelopmentLabWork

Author: Diarmuid O'Neill (C00282898@setu.ie) <br />
Date: 06/11/2025 <br />
Description: <br />
This repository demonstrates concurrency and concurrent coding concepts in GO language. It was created as part of the final year concurrent development module for the Software Development course at SETU Carlow.
Examples include:

- Atomic operations
- Mutex implementations
- Semaphore implementations
- Signalling implementation
- Barrier implementations
- Rendevous implementation
- Concurrent Fibonacci sequence implementation
- Dining Philosophers solution
- Sequential and Concurrent Game of Life implementations

GitHub Link: https://github.com/Revolution825/ConcurrentDevelopmentLabWork.git
